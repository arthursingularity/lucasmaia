💡 OPÇÃO 2: Foco Técnico e Profissional (Ideal para atrair recrutadores e outros devs)

Acabo de finalizar mais um projeto incrível: uma landing page de alta performance para a clínica do Dr. Lucas Maia! 💻✨

Neste projeto, o desafio era criar uma interface que combinasse sofisticação com altíssima performance. Para isso, escolhi uma stack moderna e robusta:

🛠️ Tecnologias utilizadas:
- Next.js (App Router) & React 19 para renderização rápida e SEO otimizado;
- Tailwind CSS v4 para uma estilização ágil e consistente;
- TypeScript para tipagem estática e código mais seguro;
- Componentização eficiente para manter o código escalável.

Destaques do projeto incluem a integração de vídeos otimizados (VSL), design totalmente responsivo e micro-interações que melhoram a experiência do usuário, como o "glow" sincronizado e o alinhamento preciso dos elementos visuais.
